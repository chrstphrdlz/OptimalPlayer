OptimalPlayer
=============

Will play a Tick Tac Toe game that will never lose. It will choose the optimal move to play against the opponet.

The Tiles are numbered:

  0   1  2
0   |  |
  --------
1   |  |
  --------
2   |  |

and the input is as follows:
Type X + Enter
Type Y + Enter
